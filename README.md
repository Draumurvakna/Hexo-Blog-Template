# My Hexo blog

## Installation

```bash
git clone https://github.com/Chen-Yulin/Hexo-Blog-src.git
cd Hexo-Blog-src
git submodule add https://github.com/Chen-Yulin/icarus.git themes/icarus
npm install -g hexo
npm install
```

## Preview

```bash
./show.sh
```

## Deploy

```bash
./deploy.sh
```

## Sync obsidian file to hexo blog

```bash
cd SyncObsidian
python3 sync.py
```
